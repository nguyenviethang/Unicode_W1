2. tim hieu the img: chen hinh anh vao trang web
cu phap: <img src="http:// duong dan anh" title=" tieu de anh khi tro chuot"/ alt="noi dung khi anh bi loi" width="chieu rong" height="chieu cao">
src: nhan duong dan anh tuong doi va tuyet doi
title: nhan tieu de
alt: nhan noi dung tieu de khi anh bi loi
width: nhan chieu rong cua anh gom:
+ value: 1 so duong (ngam hieu don vi px)
+ value: % ty le voi chieu rong cau the bao quanh nos(cap gan nhat)
+ auto: tu dong lay kich thuoc goc cua anh

luu y: moi quan he giua width va height
- neu width duoc set, height de auto hoac hong duoc set => height se tu dong  thay doi ty le voi width va nguoc lai
- ca with va height  deu duoc set=> Fix co dinh kich thuoc( de dan toi bi meo)
thuong khi lam du an se khong fix co dinh, thuong chi fix width thoi, fix chieu cao rat kho, anh huong den repoship
luu y khi dung img: anh khi set phai nho hon hoac bang anh kich thuoc goc, set to hon thi anh se vo hat(bi mo)

 img la the inline, anh se xuong hang khi kich thuoc width lon hon hoac bang 100% dua vao the cha gan nhat

 khai niem: duong dan tuong doi va tuyet doi
1. duong dan tuyet doi: protocol + domain + port + path
- protocol: giao thuc http hoac https 
- domain: ten mien(gom phan ten va phan duoi)
- port: Cong(Mac dinh 80 neu la http, 443 neu la https)
- path: duong dan
+ duong dan khu vuc
+ duong dan toi file

=> luu y : Neu file html nam trong bao nhieu cap folder => Can them bay nhieu cap ../

=> cu phap: /duong_dan
